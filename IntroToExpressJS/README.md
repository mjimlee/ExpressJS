# Set Up Process
1. Install ejs language support in VSCode
2. [Optional] Install for ease in development
```npm i --save-dev nodemon```
3. Dependencies initially set up 
```npm i```
4. To run the server, type
```npm run devStart```
5. Note that the main is server.js
